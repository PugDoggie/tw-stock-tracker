# 库存清单功能 - 5分钟快速开始

## 🚀 启动应用

### 方式1: 一键启动所有服务

```bash
npm run start:all
```

### 方式2: 分别启动

**终端1 - 启动代理服务器：**

```bash
npm run start:proxy
```

**终端2 - 启动开发服务器：**

```bash
npm run dev
```

打开浏览器访问: **http://localhost:5173**

## 📋 基本操作

### 第一步: 添加持股

1. 滚动到Dashboard中的 **"庫存組合"** 区块
2. 点击右上角 **"+ 添加"** 按钮
3. 搜索股票（输入代码或名称）
4. 输入 **成本价** (例: 500.00)
5. 输入 **持有数量** (例: 10)
6. 点击 **"確認添加"** 按钮

![Add Position Flow](docs/portfolio_add_flow.png)

### 第二步: 查看持仓和AI建议

添加后会看到：

| 列        | 说明                             |
| --------- | -------------------------------- |
| 代碼/名稱 | 股票代码和名称                   |
| 成本價    | 你的购入价格                     |
| 現價      | 当前市场价格                     |
| 數量      | 持有股数                         |
| 損益%     | 收益百分比（正数绿色，负数红色） |
| AI建議    | 系统推荐（賣出/加碼/止損/持有）  |

### 第三步: 查看详细建议

在下方 **"AI 建議詳情"** 区块查看：

- 具体的行动建议
- 建议原因
- 目标价和止损价
- 信心度分数

### 第四步: 管理持仓

**删除持股**: 点击表格右侧的 **"刪除"** 按钮

**编辑持股** (当前版本): 先删除再重新添加

## 📊 AI建议解读

### 常见的AI建议

#### 🟢 加碼 (Add More)

- **场景**: 已获利且技术面看好
- **含义**: 趁势扩大仓位
- **适用**: 看好中长期前景的投资者

#### 🔴 賣出 (Sell)

- **场景**: 已大幅获利但出现超买信号
- **含义**: 考虑获利了结
- **适用**: 想要锁定收益的投资者

#### 🟠 止損 (Stop Loss)

- **场景**: 跌幅较大且接近止损价
- **含义**: 立即止损，防止进一步亏损
- **适用**: 主动风险管理的投资者

#### 🔵 獲利出場 (Take Profits)

- **场景**: 接近目标价
- **含义**: 部分或全部获利了结
- **适用**: 稳健的获利策略

#### ⚪ 持有 (Hold)

- **场景**: 其他情况
- **含义**: 继续持有，监控变化
- **适用**: 大多数情况下的中性建议

### 信心度评分

- **🟢 80-95%**: 高度确定，强烈建议执行
- **🟡 60-79%**: 相对明确，可考虑执行
- **🔴 40-59%**: 信号混合，谨慎参考

## 📈 组合统计解读

在库存清单上方显示的4个指标：

| 指标         | 公式                            | 例子                           |
| ------------ | ------------------------------- | ------------------------------ |
| **成本總額** | 每个持股的 (成本价 × 数量) 之和 | 5,000 + 3,000 = 8,000          |
| **目前市值** | 每个持股的 (现价 × 数量) 之和   | 5,500 + 3,500 = 9,000          |
| **損益金額** | 市值 - 成本                     | 9,000 - 8,000 = +1,000         |
| **損益率**   | (損益金額 / 成本總額) × 100%    | (1,000 / 8,000) × 100 = +12.5% |

## 💾 数据保存

✅ **自动保存**: 你的持仓数据会自动保存到浏览器的localStorage

✅ **跨越刷新**: 关闭浏览器再打开，数据依然存在

⚠️ **清除会丢失**: 清除浏览器缓存/历史记录会删除数据

建议: **定期截图或记录** 重要的持仓信息

## 🎯 使用场景

### 场景1: 追踪日常持股

```
1. 早上打开应用，看到所有持股的实时价格
2. AI建议 "加碼" 某只股票时，可以考虑加仓
3. AI建议 "止損" 时，可以尽快卖出
4. 随时查看总体组合表现
```

### 场景2: 风险管理

```
1. 添加所有持股和成本价
2. 系统自动计算每个持股的损益
3. 当接近止损价时，AI会提醒
4. 可以根据AI建议及时止损，防止大亏
```

### 场景3: 获利计划

```
1. 添加持股和目标价格（在AI建议中看到）
2. 当价格接近目标时，AI建议 "獲利出場"
3. 可以分批或全部卖出
4. 记录交易结果和学习
```

## ⚙️ 设置建议

### 对于激进投资者

- 关注 "加碼" 建议，倍增获利潜力
- 信心度 ≥ 70% 时执行
- 设置止损防止大亏

### 对于稳健投资者

- 关注 "獲利出場" 建议，稳定获利
- 信心度 ≥ 80% 时执行
- 长期持有核心资产

### 对于风险规避型投资者

- 关注 "止損" 和 "賣出" 建议
- 信心度 ≥ 75% 时执行
- 优先保本，其次才是获利

## 🔧 常见问题

**Q: 可以有多个持股吗？**
A: 可以！添加无数个持股，系统会自动统计组合信息。

**Q: 价格多久更新一次？**
A: 每3秒从市场更新一次，AI建议也会相应调整。

**Q: 可以编辑成本价吗？**
A: 当前版本需要删除重新添加。后续版本会支持编辑。

**Q: 如果浏览器崩溃了，数据会丢失吗？**
A: 不会，数据存在localStorage中，重新打开浏览器就能恢复。

**Q: AI建议一定准确吗？**
A: 建议基于技术指标和历史数据，准确率在60-95%之间，不能保证100%准确。

**Q: 我想导出我的持仓数据？**
A: 当前版本没有导出功能。可以截图或手动记录，后续版本会支持。
